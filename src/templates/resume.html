Half Half Coin System (Block-Chain Development)

It allows users to create wallets first. The key pair(public key and private key) is generated when creating the wallet. Then, users can start mining the new block in order to get rewards. The meaning of mining is doing "proof of work", which is a processing of guessing the number proof. The level of mining difficulty is adjustable. Users can send cryptocurrency to other users by assigning a public key of the recipient. All transactions data will be stored in open transactions temporarily. After mining a new block, the data of open transactions is stored in the new block, and open transactions will be an empty list. Majority of the process includes different verifications in order to make sure the system is secure. All node can make HTTP requests to their peer nodes in order to share their data. (Python, HTML5, CSS, Vue).
